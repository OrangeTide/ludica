# Delta Force (needs a name)

Basic concept is the same as the A-Team. A rag tag group of veterans go on missions to help the common man.

The twist: All female military and mercenaries.

Members use code names based on tarot cards, e.g. Eight of Swords would typically just be called "Eight" by her team mates.

Game play should be retro. Over world and van driving could be like the original TMNT for NES.
Or like the overhead controls of Blaster Master.

Dialogs show character portraits. Like a more modern RPG or indie game.

To Be Determined: how is the combat. is it turn-based like an RPG. a platformer. bullet hell shooter. multiple genres at once like in Guardian Legend?
