# CC0 / Public Domain Game Fonts

Research on freely redistributable (CC0 or public domain) fonts suitable
for game development. All fonts listed are genuinely CC0/PD, not OFL/SIL.

## Selected for ludica

- **Aileron** — Clean sans-serif for UI text
- **Kenney Vector Future** — Display/title font
- **Unispace** — Monospace/technical font

## Font Sources

### Aileron (Sans-Serif, CC0)
- Author: Sora Sagano
- Style: Neo-Grotesque sans-serif, 15 weights
- Download: https://www.fontsquirrel.com/fonts/aileron
- Also at: https://usemodify.com/fonts/aileron/
- Good for: UI text, body copy, menus

### Kenney Fonts (CC0)
- Author: Kenney.nl
- Styles: KenPixel, KenPixel Square, KenPixel Blocks, KenPixel Future,
  KenVector Future (11 fonts total)
- Download: https://www.kenney.nl/assets/kenney-fonts
- Also at: https://opengameart.org/content/kenney-fonts
- Also at: https://kenney.itch.io/kenney-game-assets
- Good for: Game titles, pixel art games, sci-fi UI

### Unispace (Monospace, CC0)
- Author: Typodermic Fonts (Ray Larabie)
- Style: Technical/monospace
- Download: https://typodermicfonts.com/public-domain/
  (part of the 729-font CC0 collection)
- Good for: Debug overlays, code display, technical readouts

### Other Notable CC0 Collections

- **Typodermic Public Domain** — 729 CC0 fonts
  https://typodermicfonts.com/public-domain/
  Notable: Neuropol (sci-fi), Blue Highway (display), Unispace (mono)

- **GGBotNet Fonts CC0** — 45 CC0 fonts bundle
